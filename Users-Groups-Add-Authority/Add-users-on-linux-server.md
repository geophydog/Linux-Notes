##  新建一个名为“user” 的账户
- [x] __新建用户__  
    __`sudo useradd user -u userID -U usergroup`__
- [x] __设置密码__  
    __`sudo passwd user`__

    ### __passwd 选项：__
    __`-d  删除密码`__  
    __`-f  强制执行`__
    __`-k    设置只有在密码过期失效后，方能更新`__  
    __`-l  锁住密码`__  
    __`-s  列出密码的相关信息`__  
    __`-u  解开已上锁的账号`__
